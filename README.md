## Descrição da Atividade

Projeto desenvolvido para a atividade 04 da disciplina de HTML. O site foi criado com o tema **futebol brasileiro**, abordando times, história do esporte no país e uma página de contato.

## Páginas

- `index.html` — Página inicial com apresentação e vídeo do YouTube
- `times.html` — Informações sobre times e história do futebol brasileiro
- `contato.html` — Formulário de contato com links externos

## Tecnologias

- HTML5
- CSS3
